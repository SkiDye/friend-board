## 문제점
❌ 게시물에 URL을 붙여넣어도 텍스트로만 표시
❌ 링크를 클릭할 수 없어 복사-붙여넣기 필요
❌ 공유하기 불편함

## 해결책 - Linkify 기능 구현

### URL 자동 감지 및 변환
```javascript
const linkifyText = (text) => {
  const urlRegex = /(https?:\/\/[^\s]+)/g

  // URL 찾아서 <a> 태그로 변환
  return text.split(urlRegex).map((part, i) => {
    if (urlRegex.test(part)) {
      return (
        <a href={part} target="_blank" rel="noopener noreferrer"
           className="text-blue-600 hover:text-blue-800 underline">
          {part}
        </a>
      )
    }
    return part
  })
}
```

## 작동 방식

**Before**
```
추천 사이트: https://example.com
```
→ 그냥 텍스트 (클릭 불가)

**After**
```
추천 사이트: https://example.com
              ↑ 파란색 밑줄 (클릭 가능!)
```
→ 클릭하면 새 탭에서 열림

## 특징

✅ **자동 감지**: http:// 또는 https://로 시작하는 모든 URL
✅ **새 탭 열기**: `target="_blank"` (현재 페이지 유지)
✅ **보안 강화**: `rel="noopener noreferrer"`
✅ **시각적 구분**: 파란색 밑줄 스타일
✅ **호버 효과**: 마우스 올리면 더 진한 파란색
✅ **유튜브 우선**: 유튜브 URL은 기존대로 플레이어로 표시

## 사용된 기술

- **정규 표현식**: `/(https?:\/\/[^\s]+)/g`
- **React 조건부 렌더링**: URL이면 링크, 아니면 텍스트
- **Tailwind CSS**: `text-blue-600`, `hover:text-blue-800`, `underline`
