## 내용 (아래 전체를 복사)

🖼️ **더 넉넉한 여백으로 가독성 향상**

## 문제점
❌ 이미지/동영상이 화면 가장자리에 너무 붙어있음
❌ 콘텐츠 간 간격이 좁아 답답함
❌ 모바일에서 특히 여유 공간 부족

## 해결책
```javascript
// ImageViewer.jsx
<div className="px-6 sm:px-8 md:px-12 py-3">
  <LazyImage ... />
</div>
```

## 여백 변경

| 화면 크기 | 좌우 여백 | 상하 여백 |
|---------|---------|---------|
| 📱 모바일 | 16px → **24px** | 8px → **12px** |
| 📱 태블릿 | 24px → **32px** | 8px → **12px** |
| 💻 데스크톱 | 32px → **48px** | 8px → **12px** |

## 사용된 기술
- **Tailwind CSS**: 반응형 유틸리티 클래스
- **Responsive Design**: 화면 크기별 최적화

## 개선 결과
✅ 미디어와 화면 가장자리 간 여유 공간 확보
✅ 이미지 간 구분이 명확해짐
✅ 더 고급스럽고 편안한 레이아웃
✅ 모바일에서 스크롤 시 손가락 공간 확보

→ 작은 변화지만 체감 사용성 대폭 개선
